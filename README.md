# psdtool

`npm install -g rstacruz/psdtool`

```sh
$ psdtool file.psd --fonts

# file.psd
fonts:
  - DIN-Bold
  - FreightSansLight
  - Glosa-Roman
  - ...
```

```sh
$ psdtool file.psd --text

# file.psd
text:
  - "Hello"
  - "This is text from the document"
  - ...
```
